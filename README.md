## HR Analytics Dashboard (Tableau)
This project focuses on analyzing employee data using Tableau to support data-driven HR decisions.

## Project Overview

Built an interactive HR analytics dashboard to study workforce demographics, salary distribution, employee growth, education impact, and leave patterns.

## Dataset Details
.Employee data including job role, gender, age, salary, education, and leave balance
.Time period covered: 2017 to 2023

## Key Insights
.The 30–40 age group represents around 50% of the workforce
.Female employees (88) are higher than male employees (73)
.Employees with a master’s degree earn higher salaries
.Sales and Marketing roles show higher leave balances
.Steady employee growth observed from 2017 to 2023

## Tools Used
Tableau for data visualization and dashboard creation

## Outcome
The dashboard helps HR teams understand workforce trends, improve planning, and make informed decisions related to hiring, compensation, and employee management.
